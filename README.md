# VELVET contactos · Acceso Exclusivo VIP

Imagen de marca y dirección de arte para la app **VELVET_contactos**, desplegada en GitHub Pages.

- **URL pública:** https://sandrosiliuto.github.io/velvet-contactos/
- **Tagline:** *EN LA VIDA TODO SON CONTACTOS*
- **Estado:** VIP siempre visible.

## Identidad aplicada

- Paleta de lujo nocturno: Onyx Black, Velvet Mauve, Rose Gold y Brut Champagne.
- Tipografías Cinzel (serif) + Inter (sans).
- Glassmorphism, gradiente metálico oro rosa y destellos estelares.
- Componentes: Welcome, tarjeta de perfil, detalle de usuario, chat y CTA VIP.

## Uso local

Abrir `index.html` directamente en el navegador. No requiere build.
